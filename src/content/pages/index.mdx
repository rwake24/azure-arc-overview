    <h3>Why Azure Arc?</h3>
    <ul>
      <li><strong>Single control plane</strong> — Manage servers, Kubernetes clusters, and data services across on-premises, multicloud, and edge from the Azure portal.</li>
      <li><strong>Consistent governance</strong> — Apply Azure Policy, RBAC, and tagging to all resources regardless of where they run.</li>
      <li><strong>Cloud-native everywhere</strong> — Bring Azure PaaS data services, GitOps, and developer tooling to any Kubernetes cluster.</li>
      <li><strong>Security at scale</strong> — Extend Microsoft Defender for Cloud, Sentinel, and compliance baselines to your entire hybrid estate.</li>
      <li><strong>No migration required</strong> — Get Azure benefits without moving workloads. Meet data residency and latency requirements while using Azure management.</li>
    </ul>
